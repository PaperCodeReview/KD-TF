# KD-TF

This is an unofficial implementation of Knowledge Distillation ([Distilling the Knowledge in a Neural Network, NIPS 2014 Deep Learning Workshop](https://arxiv.org/abs/1503.02531)).

## Requriments
- tensorflow >= 2.x

## Training
```
python main.py
```
